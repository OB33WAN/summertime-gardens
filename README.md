# Summertime Gardens static website

This is a plain HTML, CSS and JavaScript site suitable for GitHub Pages.

## Files

- `index.html` - page markup, SEO meta tags and local business schema
- `styles.css` - responsive visual design
- `script.js` - mobile menu and email-based enquiry form
- `assets/` - compressed image assets and favicon
- `robots.txt` and `sitemap.xml` - search engine files

## Before launch

Update these placeholders:

- Phone number: `020 0000 0000` and `+442000000000`
- Email address: `hello@summertimegardens.co.uk`
- Canonical, Open Graph, sitemap and robots URLs if the final domain is different from `https://summertimegardens.co.uk/`

For GitHub Pages, copy these files to the repository root or the configured Pages folder and publish from that branch.
